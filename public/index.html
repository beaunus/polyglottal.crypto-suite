Caesar Cipher

The Caesar Cipher is a substitution cipher in which each letter of the message is shifted to a corresponding letter that is some number of letters away. For example, if the letter "E" were shifted to the left by 3, the result would be the letter "B".

Scytale Cipher

The Scytale Cipher is a transposition cipher in which each letter of the message is moved to a different position in the message. A scytale is a cylindrical rod around which a piece of paper or leather is wound. The message is written across the paper. When the paper is unwound, the letters of the message are transposed.

https://en.wikipedia.org/wiki/Scytale
